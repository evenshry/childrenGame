# 指令大冒险 - 编程启蒙游戏

一个面向儿童和编程初学者的可视化编程教育游戏，通过拖拽指令块控制机器人完成各种任务。

## 技术栈

- **前端框架**：React 18 + TypeScript
- **UI 组件库**：Ant Design 5
- **样式方案**：Sass + CSS Modules
- **状态管理**：Zustand（支持持久化）
- **路由管理**：React Router v6
- **拖拽功能**：React DnD
- **构建工具**：Vite
- **代码规范**：ESLint + Prettier

## 核心功能

### 🎮 游戏模式
- **拖拽编程**：通过拖拽指令块编写程序
- **实时预览**：即时查看机器人执行效果
- **调试模式**：单步执行、设置断点、查看执行日志
- **关卡系统**：包含多个精心设计的教学关卡

### 📚 指令类型
| 指令类型 | 描述 |
|---------|------|
| 基础指令 | 前进、左转、右转、跳跃、瞬移 |
| 循环指令 | 循环 N 次、重复直到条件满足 |
| 条件指令 | 如果前面被阻挡、如果左侧被阻挡、如果右侧被阻挡 |
| 特殊指令 | 收集物品、放置物品、对话、随机转向 |

### 🎯 特性
- **撤销/重做**：支持无限撤销和重做操作
- **程序管理**：保存和加载程序方案
- **自定义关卡**：创建和编辑自定义关卡
- **成就系统**：完成挑战解锁成就
- **响应式设计**：适配桌面和移动设备


```bash

```

### 构建生产版本
```bash

```

### 代码规范检查
```bash
npm run lint
```

## 设计原则

1. **类型优先**：所有 API、Props、State 必须有 TypeScript 类型
2. **样式隔离**：使用 CSS Modules + 全局变量，避免全局污染
3. **组件复用**：Ant Design 为基础，业务组件按需封装
4. **状态可控**：Zustand 管理全局状态，组件状态管理局部状态
5. **性能意识**：懒加载、useCallback、useMemo 优化
6. **规范落地**：ESLint + Prettier 保证代码质量
